# lain alien themes
this is a small but possibly growing repo of dumb little pixel arts
they go around ur windows on ur linux desktop, probably just xfce

## installing
just put the namd theme folder containing the xfwm4 folder in one of
```
~/.local/share/themes
~/.themes
/local/share/themes
```
